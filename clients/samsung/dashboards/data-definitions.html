Primary KPIs

The 4 KPI cards displayed at the top of the GEO Dashboard.

Share of Voice

KPI
Description
Samsung's percentage of total brand mentions across all AI models. Higher values indicate stronger brand presence in AI-generated responses. This metric shows how often Samsung is mentioned compared to competitors like LG, Sony, TCL, and Hisense.
Calculation
(Samsung_mentions / Total_brand_mentions) × 100
Data Source
Semrush semrush_concept_mentions via RPC

Visibility Score

KPI
Description
Normalized 0-100 score representing Samsung's visibility relative to total mentions across all AI models. Uses a 1.5x multiplier to scale raw percentages into a more meaningful score. A score of 100 represents maximum possible visibility.
Calculation
MIN(100, (Samsung_mentions / Total_mentions) × 100 × 1.5)
Data Source
Semrush semrush_concept_mentions via RPC

Sentiment Score

KPI
Description
Net sentiment expressed as a percentage. Positive scores (e.g., +15) indicate more favorable coverage; negative scores indicate more critical coverage. A score of 0 means equal positive and negative mentions.
Calculation
(positive - negative) / (positive + negative + neutral) × 100
Data Source
Semrush sentiment_positive, sentiment_negative, sentiment_neutral fields

Total Mentions

KPI
Description
Count of all Samsung-related concept mentions in AI responses across all tracked prompts and models. This is the raw volume of times Samsung appears in AI-generated content.
Calculation
SUM(total_mentions) WHERE brand = 'Samsung'
Data Source
Semrush semrush_concept_mentions

Domain Sentiment Analysis

Metrics for understanding how different source domains discuss Samsung in AI responses.

Sentiment by Domain

Metric
Description
Positive, neutral, and negative mention counts for each source domain cited in AI responses. Shows which domains (e.g., rtings.com, techradar.com) are most favorable or critical of Samsung.
Calculation
COUNT(*) GROUP BY domain, sentiment
Data Source
Semrush semrush_concept_prompts

Domain Type Breakdown

Metric
Description
Distribution of mentions across source type categories:
  • Owned: samsung.com domains
  • Earned: Review sites, tech publications (rtings.com, techradar.com)
  • Social: Reddit, YouTube, Twitter/X
  • Competitor: lg.com, sony.com, tcl.com, hisense.com
  • Other: All other domains
Calculation
SUM(mentions) GROUP BY domain_type
Data Source
Semrush via domain classification logic

Domain Performance Summary

Metric
Description
Table showing each domain's total mentions, unique URLs cited, and sentiment breakdown (positive, neutral, negative counts). The "Neg %" column shows what percentage of that domain's coverage is negative. Used to identify which sources are most/least favorable.
Calculation
Aggregated per-domain: unique_urls, positive, neutral, negative, negative_pct = negative/(positive+neutral+negative)
Data Source
Semrush semrush_concept_prompts aggregated

Citation Metrics

Metrics from the Citation Sources and Cited URLs tables.

Citations

Metric
Description
Number of times a domain or URL is cited as a source in AI-generated responses. When an AI response includes a link to a domain, that counts as one citation. More citations means the domain is used more frequently as a reference.
Calculation
COUNT(DISTINCT prompt_url pairs) or prompts_count field
Data Source
Semrush semrush_prompt_urls, semrush_cited_pages

Mentions (in Citation context)

Metric
Description
Total mention count for a domain across all prompts where it appears as a cited source. Different from the primary KPI "Total Mentions" - this is domain-specific.
Calculation
SUM(mentions_end) WHERE domain = x
Data Source
Semrush Source Visibility API

Visibility % (Citation)

Metric
Description
Percentage of all AI responses (for tracked prompts) that cite this domain as a source. A domain with 10% visibility appears as a source in 10% of all AI responses.
Calculation
(domain_citations / total_citations) × 100 or visibility field from API
Data Source
Semrush visibility field

URLs Cited

Metric
Description
Count of unique URLs from a specific domain that appear in AI responses. A domain with many cited URLs has diverse content being referenced.
Calculation
COUNT(DISTINCT url) WHERE domain = x
Data Source
Semrush semrush_cited_pages

Concept Analysis

Metrics for understanding what features, attributes, and topics are discussed about Samsung.

Concept Mentions

Metric
Description
Number of times a specific concept (e.g., "picture quality", "input lag", "HDR", "gaming performance") appears in AI responses about Samsung. Concepts are product features, attributes, or topics that Semrush's AI has identified.
Calculation
SUM(mentions_end) WHERE concept = x AND brand = 'Samsung'
Data Source
Semrush semrush_concept_mentions

Top Positive Concepts

Metric
Description
Most frequently mentioned concepts with positive sentiment. Indicates what Samsung is praised for in AI-generated content. Use these concepts in marketing messaging.
Calculation
ORDER BY positive_count DESC WHERE brand = 'Samsung'
Data Source
Semrush sentiment fields aggregated by concept

Top Negative Concepts

Metric
Description
Most frequently mentioned concepts with negative sentiment. Indicates areas of criticism in AI-generated content. These are topics that may require PR attention or product improvements.
Calculation
ORDER BY negative_count DESC WHERE brand = 'Samsung'
Data Source
Semrush sentiment fields aggregated by concept

Concept Sentiment Mix

Metric
Description
Stacked bar visualization showing the positive/neutral/negative distribution for each concept. Helps identify concepts with mixed sentiment (controversial) vs. consistently positive or negative coverage.
Calculation
(positive, neutral, negative) / total_mentions per concept
Data Source
Semrush sentiment fields

Platform Metrics

Performance metrics broken down by AI model/platform.

AI Models Tracked
search-gpt → ChatGPT (SearchGPT) google-ai-overview → Google AI Overview google-ai-mode → Google AI Mode

Platform Mentions

Metric
Description
Samsung mention count per AI model. Shows which AI platforms mention Samsung most frequently. Useful for understanding where Samsung has strongest/weakest presence.
Calculation
SUM(total_mentions) WHERE model = x AND brand = 'Samsung'
Data Source
Semrush semrush_concept_mentions

Platform Trend

Derived
Description
Change in mentions compared to previous period, broken down by AI model. Shows which platforms are growing/declining in Samsung coverage. Displayed as percentage change in the change badges.
Calculation
(current_period - previous_period) / previous_period × 100
Data Source
Calculated from time-series data

Trend Metrics

Time-series metrics for tracking changes over time.

Mentions Over Time

Metric
Description
Daily or weekly count of Samsung mentions across all AI models. The trend line chart shows how mention volume changes over time, with separate lines for Samsung and competitors.
Calculation
SUM(total_mentions) GROUP BY date WHERE brand = 'Samsung'
Data Source
Semrush semrush_concept_mentions via get_daily_mentions RPC

Period Comparison

Derived
Description
Percentage change between current selected period and previous period of equal length. For example, if viewing "Last 30 days", compares to the 30 days before that. Shown in the change badges below each KPI.
Calculation
(current_sum - previous_sum) / previous_sum × 100
Data Source
Calculated in dashboard JavaScript

PR Action Center

Metrics for identifying issues requiring PR attention and response.

Negative Quotes

Metric
Description
Direct quotes extracted from AI responses with negative sentiment about Samsung. These are the actual text snippets that AI models are generating when being critical of Samsung products. Useful for identifying specific criticisms that need addressing.
Calculation
SELECT quote, prompt, concept, date WHERE sentiment = 'negative' AND brand = 'Samsung'
Data Source
Semrush semrush_concept_prompts

Negative Concept Frequency

Metric
Description
How often each concept appears in negative coverage. Prioritizes which issues to address first. The "Top Negative Concepts" chart shows concepts ordered by negative mention count.
Calculation
COUNT(*) WHERE sentiment = 'negative' GROUP BY concept ORDER BY count DESC
Data Source
Semrush semrush_concept_prompts

Prompts with Negative Quotes

Metric
Description
Which user prompts/questions are generating the most negative responses about Samsung. Helps identify the types of questions that lead to unfavorable AI outputs.
Calculation
COUNT(*) WHERE sentiment = 'negative' GROUP BY prompt ORDER BY count DESC
Data Source
Semrush semrush_concept_prompts

Understanding the Data

Key concepts to understand when interpreting dashboard metrics.

Mentions vs Citations
  • Mention: When Samsung is discussed in an AI response. The brand is talked about, referenced, or recommended. Example: "Samsung TVs are known for their picture quality."
  • Citation: When samsung.com (or any domain) is linked as a source in an AI response. The brand's content is being used as a reference. Example: "According to samsung.com..."
Sentiment Classification
  • How it works: Semrush AI analyzes the tone of AI responses discussing Samsung
  • Positive: Favorable language, recommendations, praise. Example: "Samsung offers excellent HDR performance"
  • Negative: Criticism, warnings, unfavorable comparisons. Example: "Samsung's gaming mode has noticeable input lag"
  • Neutral: Factual statements without opinion. Example: "Samsung TVs range from $500 to $5000"
Data Freshness
  • Data is loaded daily from Semrush Enterprise API
  • Date range filters in the dashboard apply to the selected period
  • Historical data is retained for trend analysis
Position/Ranking
  • Position 1 = first brand/product mentioned in the AI response, or first citation
  • Lower position numbers are better (appearing earlier = more prominent)
  • Position affects visibility weighting in some calculations
AI Model Coverage
  • ChatGPT (SearchGPT): OpenAI's conversational AI with web search
  • Google AI Overview: AI summaries appearing in Google search results
  • Google AI Mode: Google's conversational AI experience
  • Not yet tracked: Perplexity, Claude, Copilot, Gemini

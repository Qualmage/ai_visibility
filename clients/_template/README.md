# Client Template

Copy this folder and rename it to create a new client workspace.

## Folder Structure

```
client-name/
├── gtm/           # Google Tag Manager configs, exports, documentation
├── ga/            # Google Analytics reports, queries, documentation
├── looker/        # Looker Studio dashboards and data sources
├── docs/          # Client-specific documentation
└── README.md      # Client overview and key info
```

## Getting Started

1. Copy this `_template` folder
2. Rename to client name (lowercase, hyphens): `acme-corp`
3. Update the README.md with client details
4. Add GTM container IDs and GA property IDs to the README
